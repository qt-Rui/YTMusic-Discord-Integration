Why you'll like it

🔒

Private by design

No telemetry, no accounts, no remote servers. The bridge binds to 127.0.0.1 only and rejects anything off-machine.

🎨

Rich, accurate presence

High-quality album art and a progress bar that stays in sync with YouTube Music — including pause and seek.

♻️

Reliable & light

Auto-reconnects to Discord, throttles updates, and lives quietly in your system tray. Can launch on startup.

🌐

Works everywhere

YouTube Music in any Chromium browser or Firefox, on Windows, macOS, and Linux.

Get started in 4 steps

  1. Install the desktop app

    Download above and run the installer. Make sure the Discord desktop app is running (the web version can't be used for Rich Presence).

  2. Load the browser extension

    The installer includes an extension folder (in the app, click Setup next to the YouTube Music row to reveal it). In Chromium: open chrome://extensions, enable Developer mode, click Load unpacked, and select that folder. Firefox users: see the extension's README.

  3. Add your Discord application ID

    Create a free app at discord.com/developers/applications, copy its Application ID, and paste it into Settings → Advanced. Its name becomes your "Listening to …" label.

  4. Press play 🎉

    Open music.youtube.com, start a song, and your Discord profile lights up.

📖 Full walkthrough and troubleshooting live in the project README.

Privacy & security

🚫

Nothing leaves your machine

All processing is local. The only connection the extension makes is the loopback WebSocket to the desktop app.

🔍

Open source & auditable

Every line is on GitHub under the MIT license. Read it, build it yourself, or send a pull request.

🧩

Minimal extension access

The extension can only touch music.youtube.com and loopback — no broad browsing access.
