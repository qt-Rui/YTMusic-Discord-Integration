# resources/

`icon.png` is generated by `scripts/make-icon.mjs` (run `npm run make:icon`) and
is used by electron-builder to produce the platform app icons (`.ico` / `.icns`
are derived from it automatically during packaging).

To use your own icon, replace `icon.png` with a square PNG of at least
512×512 px (1024×1024 recommended). The tray/window icon at runtime is drawn in
code (`src/main/icon.ts`) and does not depend on this file.
